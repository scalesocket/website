What is ScaleSocket?

Scalesocket is a websocket server and autoscaler. It servers your application via websockets and scales the number of instances based on the number of connections.

The main use-case of ScaleSocket is to vastly simplify the structure of realtime web apps by moving the connection management and scaling outside the application code.
